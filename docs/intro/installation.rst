Installation
============

Make sure you have the :doc:`system-requirements` installed.  You will need Python, pip, virtualenv, and virtualenvwrapper.

Installation
------------

To install **gthnk**, open a terminal and type the following:

::

    mkvirtualenv gthnk
    pip install gthnk
    install_osx.sh

Creating a User
^^^^^^^^^^^^^^^

Installation Files
^^^^^^^^^^^^^^^^^^

- virtualenv
- ~/Library/Gthnk

Uninstallation
--------------

::

    workon gthnk
    uninstall_osx.sh
